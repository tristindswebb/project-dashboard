🚀 Setup Instructions

  1. Create a Google Sheet with these exact column headers:
    Task Name | Owner | Priority | Status | Start Date | End Date | Progress | Description
  2. Share your sheet: File → Share → Change to "Anyone with the link can view"
  3. Add your sheet ID to GitHub Secrets
  4. Deploy using GitHub Actions
🔒 Security Benefits:
  • Sheet URL is hidden in environment variables
  • Only authorized team members can edit the sheet
  • Viewers see polished dashboard, not raw data
